## 三、如何新增/更换口令

### 方法：在线生成（推荐）

如果你需要生成新的口令，可以直接在浏览器控制台执行：

```javascript
// 在浏览器 F12 控制台执行
async function sha256(text) {
  const buf = await crypto.subtle.digest('SHA-256', new TextEncoder().encode(text));
  return Array.from(new Uint8Array(buf)).map(b => b.toString(16).padStart(2,'0')).join('');
}
// 生成你的口令哈希
sha256("ZT940-你的新口令").then(console.log);
```

### 方法：命令行生成

```bash
python3 -c "import hashlib; print(hashlib.sha256('ZT940-你的新口令'.encode()).hexdigest())"
```

### 替换到代码中

打开 `assets/member-paywall.js`，找到 `ACTIVATION_CODE_HASHES` 数组，添加新哈希：

```js
var ACTIVATION_CODE_HASHES = [
  // ... 已有口令
  '你的新口令哈希'
];
```

**注意：**
- 可以同时保留多个口令，旧口令不会失效
- 如果想让某个口令失效，直接删除对应的哈希行即可
- 用户换浏览器或清缓存后需要重新输入口令

## 四、推荐人工收款流程

1. 用户打开网页，点击 `会员登录 / 开通`。
2. 用户扫码支付 `299 元`。
3. 用户付款备注填写手机号或微信号。
4. 你在支付宝账单里核对到账。
5. 你把当月开通口令发给用户。
6. 用户输入口令后，解锁 9:40 模块 + 次日跟踪模块 30 天。

## 五、必须知道的限制

这个方案不接支付 API，不需要服务器，因此免费、简单，但安全性有限：

- 技术用户仍可能通过浏览器绕过前端门禁。
- 不能自动核验支付宝是否到账。
- 不能自动续费。
- 不能统一管理会员。
- 用户换浏览器或清缓存后，本机会员状态会丢失，需要重新输入口令。

如果开始真实收费并有较多用户，建议升级为：

```text
Cloudflare Pages
+ Cloudflare Workers
+ D1 数据库
+ 支付宝当面付 / 微信支付 Native
+ 服务端会员鉴权
```
